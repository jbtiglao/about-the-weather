# About the Weather

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

  ---
  ## Description

 This is a weather dashboard that retrieves weather data for cities, with current weather condition and five-day forecast using OpenWeather API. It runs in a browser and features dynamically updated HTML and CSS.
